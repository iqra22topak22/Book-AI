# Static Assets for Physical AI Course

This directory contains static assets for the Physical AI & Humanoid Robotics course documentation.

## Directory Structure
```
static/
├── img/
│   ├── robot-arm.svg
│   ├── simulation.svg
│   ├── hands-on.svg
│   └── course-diagrams/
└── ...
```

## Image Guidelines
- All images should be optimized for web (compressed)
- SVG format preferred for diagrams and logos
- PNG format for screenshots and photos
- Images should have meaningful names that reflect their content
- Diagrams should be stored in the course-diagrams subdirectory

## Creating New Assets
When creating new assets, consider:
1. Accessibility: Include descriptive alt text
2. Responsiveness: Ensure assets look good on all screen sizes
3. File size: Optimize images to minimize load times
4. Licensing: Ensure all assets comply with appropriate licenses
